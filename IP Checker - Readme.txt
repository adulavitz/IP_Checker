# IP Checker
> Lightweight network monitor for real-time external IP and ASN tracking

Created by:	Andrew J. Dulavitz
Contact: 	adulavitz@hotmail.com
Date:		June 2025

IP Checker scans every 30 seconds to determine your PCâ€™s external IP address and ASN (Autonomous System Number). It provides real-time network information and logs all activity locally.

## Features

- Displays the name of the connected network
- Shows your current external IP address
- Identifies the ASN and provider name
- Logs connection events with timestamps (monthly logs)
- Detects and displays connection type (wired/wireless)
- GUI indicator shows active connection status
- Icons reflect connection type
- Supports dark mode
- Minimizes to the system tray
- Quick-access button opens the log file folder
- Plays customizable audio alerts on connection events
- Restores from tray on connection changes
- Low resource usage (~36MB memory)
